# Smart Home Dashboard 🏠

## Objetivo 🎯
Diseñar una **Landing Page de un Dashboard de Control para una Smart Home** que permita visualizar y gestionar diferentes elementos del hogar mediante widgets como temperatura, luces, consumo energético y cámaras.

El dashboard funciona como **aplicación móvil y panel de control para escritorio o tablet**.

## Funcionalidades ⚙️

Widgets incluidos:

- 🌡 Temperatura  
- 💡 Luces  
- ⚡ Consumo energético  
- 📷 Cámaras  
- 🔒 Seguridad  
- 🌬 Calidad del aire  
- ⚙ Rutinas  
- 🚪 Puertas  

Incluye **modo día y modo noche**.

## Tecnologías 💻

- HTML  
- CSS   
- Figma
- Git 
- GitHub 

## ¿Cómo ejecutarlo? 🚨

1. Clonar el repositorio.
2. Abrir el archivo index.html en el navegador.

El dashboard se cargará automáticamente.
